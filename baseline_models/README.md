# Baseline Models

For all models below the `dataset_type` parameter has three possible options:

1. toyota
2. foursquare
3. gowalla

## ST-RNN

```
cd STRNN/
python train.py --dataset_type <dataset_type>
```

## LSTPM
```
cd LSTPM/
python train.py --dataset_type <dataset_type>
```

## STAN
```
cd STAN/
python train.py --dataset_type <dataset_type>
```
